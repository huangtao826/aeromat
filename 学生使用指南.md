# Aeromat AI 学生使用指南

## 快速开始

### 步骤1：检查 Python
如果没有安装 Python，先下载安装：
- 下载地址：https://www.python.org/downloads/
- **安装时务必勾选：Add Python to PATH**

### 步骤2：运行
1. 双击 `run.bat`
2. 首次运行会自动安装依赖（约1-2分钟）
3. 浏览器自动打开 http://localhost:8501

### 步骤3：配置 LLM（可选）
如果没有 API Key，智能体只能使用内置知识库回答。

如需更智能的回答：
1. 获取通义千问 API Key（免费额度）
2. 在 Windows 命令行执行：
   ```
   set DASHSCOPE_API_KEY=你的密钥
   run.bat
   ```

## 常见问题

### Q: 双击 run.bat 后窗口一闪而过？
A: 说明 Python 未安装或未添加到 PATH。请先安装 Python。

### Q: 提示"不是内部或外部命令"？
A: Python 未正确安装。重新安装 Python，确保勾选 "Add Python to PATH"。

### Q: 启动后浏览器没有自动打开？
A: 手动打开浏览器，访问 http://localhost:8501

### Q: 如何更新知识库？
A: 在 `knowledge/docs/` 或 `knowledge/faqs/` 文件夹中添加 .md 文件即可。
