### Q: 在Procast中如何增加冷却速度

### A: 在ProCAST中增加冷却速度主要通过调整**界面换热系数（Heat Transfer Coefficient, h）**、**添加冷铁（Chill）**、**降低模具/环境温度**以及**优化冷却方式**来实现。

### 以下是具体操作方法：

## 一、核心参数：界面换热系数（h值）

界面换热系数是控制冷却速度的最直接参数，**h值越大，冷却速度越快**。

### 典型设置参考：

| 接触界面类型             | 推荐h值 (W/m²·K)      |
| ------------------------ | ----------------------- |
| 铸件与砂型               | 300~1000（通常取500）   |
| 铸件与冷铁               | 1000~5000（通常取2000） |
| 铸件与金属型（高压铸造） | 3000~3500               |
| 砂型与冷铁               | 500左右                 |

**设置路径**：在Visual-Cast或PreCAST中，通过 **Interface** 菜单定义不同材料接触面的换热系数。

> **注意**：设置完Type后需点击 `Apply`按钮确认，否则参数不会生效。

## 二、添加冷铁（Chill）

在铸件热节或需要加速冷却的部位放置冷铁是工程上最常用的方法：

- **冷铁材料**：铸铁、铜等导热性好的金属
- **厚度建议**：一般为被激冷部位铸件壁厚的0.8~1倍
- **效果**：铸件与冷铁之间的h值通常设置为**2000 W/(m²·K)**，远高于砂型

**案例**：在铝合金轮毂中心安装面放置外冷铁，轮辐位置设计间接外冷铁，可显著加速局部冷却，消除缩松缺陷。

## 三、调整工艺参数

### 1. 降低模具/砂型初始温度

- 模具温度越低，冷却速度越快
- 例如：铝合金轮毂模具温度从410°C降至268°C，冷却速率明显提高（但需兼顾充型质量）

### 2. 降低浇注温度

- 浇注温度越低，金属液带入的热量越少，整体冷却加快
- 通常浇注温度高于液相线100°C左右，可适当降低但避免冷隔缺陷

### 3. 冷却方式设置

在边界条件（Boundary Conditions）中：

- **空冷（Air Cool）**：默认自然冷却，速度较慢
- **强制冷却**：可设置更高的换热系数模拟风冷或水冷
- **水冷**：在离心铸造等工艺中，水冷阶段可大幅加速冷却（如70秒水冷阶段）

## 四、多阶段冷却控制（高级）

对于复杂工艺，可使用ProCAST的多工序流程模板：

1. **铸型移除阶段**：移除保温层后散热加快
2. **强制冷却阶段**：设置不同的h值模拟水冷或风冷
3. **炉体辐射控制**：在熔模铸造中，通过设置高温区（1550°C）、中温区（900°C）、低温区（20°C）的辐射系数来调节冷却环境

## 五、验证冷却效果

在后处理（PostCAST/ViewCAST）中查看：

- **Cooling Rates**：显示指定温度区间的冷却速率（°C/s）
- **Solidification Time**：凝固时间越短说明冷却越快
- **Niyama Criterion**：辅助判断缩孔风险，冷却速度增加通常会改善补缩条件

## 总结建议

| 目标             | 推荐操作                                    |
| ---------------- | ------------------------------------------- |
| 全局加速冷却     | 降低模具初始温度、适当降低浇注温度          |
| 局部加速冷却     | 在热节处添加冷铁（h=2000）                  |
| 精确控制冷却曲线 | 使用温度相关的h值函数（液态2000→固态1700） |
| 消除缩松缩孔     | 冷铁+冒口组合，优化凝固顺序                 |

**关键原则**：增加冷却速度需确保**顺序凝固**，即铸件薄壁处先凝固，厚壁/热节处后凝固，避免孤立液相区导致缩孔。
