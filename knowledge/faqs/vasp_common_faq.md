# VASP 计算常见问题 FAQ

## Q1: 如何选择ENCUT值？

**A:** ENCUT应设置为材料最大ENMAX的1.2-1.3倍。可通过以下命令查看：
```
grep ENMAX POTCAR
```
推荐值：520-600 eV（对于普通体系）

---

## Q2: 能带计算结果带隙为0？

**可能原因：**
1. 泛函选择不当（LDA通常低估带隙）
2. 未进行自洽计算
3. k点路径设置错误

**推荐解决方案：**
- 尝试GGA-PBE泛函
- 使用HSE06杂化泛函（计算成本较高）
- 检查KPOINTS路径是否正确

---

## Q3: 计算内存不足？

**解决方案：**
1. 降低K点密度（如11 11 11 → 8 8 8）
2. 减少平面波数量（降低ENCUT）
3. 使用PAW赝势替代 ultrasoft
4. 增加节点数或每节点核数
